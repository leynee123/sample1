LINUX CLI BASICS COMMAND
pwd = "show me the folder I'm currently in"
ls = what files or folders are here; ls -l = more details; ls -al = view also hidden files
cd <directory> = where u want to go; cd.. = back one level
find <starting_point> -name <filename>; starting point = ~ means begin home directory
cat <filename> = get the content of the file

uname -a = see details of  operating system, kernel version, and architecture
df -h = check disk usage
/etc directory* = Linux stores configuration and informational files
